/*
  DigitalReadSerial

  Reads a digital input on pin 2, prints the result to the Serial Monitor

  This example code is in the public domain.

  http://www.arduino.cc/en/Tutorial/DigitalReadSerial
*/

// digital pin 2 has a pushbutton attached to it. Give it a name:
int pushButton = 2;

// the setup routine runs once when you press reset:
void setup() {
  // initialize serial communication at 9600 bits per second:
  Serial.begin(9600);
  // make the pushbutton's pin an input:
  pinMode(pushButton, INPUT);
}

// the loop routine runs over and over again forever:
void loop() {
  // read the input pin:
  int buttonState = digitalRead(pushButton);
  // print out the state of the button:
  Serial.println(buttonState);
  delay(1);        // delay in between reads for stability
}

El en loop() encontramos la primera una estructura de control

5 - if

Descripción

La estructura de control if verifica si se cumple una condición y ejecuta una declaración o el conjunto de declaraciónes.

Sintaxis

if (condición) {//declaración(s)}

Parámetros

(condición): una expresión booleana (es decir, puede ser verdadero o falso).

Otra estructura de control:

6 - else

Descripción

Con la estructura de control else se puede proceder con otra prueba if de modo que se puedan ejecutar múltiples pruebas, si ninguna prueba resulta ser verdadera, else ejecuta un bloque predeterminado.

else

Sintaxis

else if (condición 2) {//declaración(s)}

Parámetros

Igual que if.

Podemos probar con motores, buzzers y otros actuadores que admitan valores digitales.
